# PixelCraft AI — Free Online Image Tools

PixelCraft AI is a suite of professional, browser-based image editing tools. All processing happens locally on your device, ensuring maximum privacy and speed. No sign-up, no server uploads, 100% free.

## ✨ Features
- **AI Background Remover**: Effortlessly remove backgrounds using state-of-the-art AI.
- **AI Object Remover**: Erase unwanted objects from your photos.
- **Smart Social Resize**: Automatically resize images for Instagram, YouTube, LinkedIn, and more.
- **Image Compressor**: Reduce file size without losing quality.
- **Format Converter**: Convert between PNG, JPG, WebP, and AVIF.
- **Filters & Effects**: Apply professional-grade adjustments and artistic filters.
- **Privacy First**: Your images never leave your browser.

## 🚀 Built With
- **Core**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **AI Engine**: Transformers.js (Running RMBG-1.4 Locally)
- **Styling**: Modern CSS Grid and Flexbox for responsiveness.

## 🛠️ Installation & Local Usage
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Open `index.html` in any modern browser OR run a local server:
   ```bash
   npx serve .
   ```

## 🌐 Deployment
This site can be easily deployed to **GitHub Pages**, **Vercel**, or **Render** as a static website.
